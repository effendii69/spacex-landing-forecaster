# SpaceX Falcon 9 Landing Success Forecaster

Live-updating web app that predicts the probability of the next Falcon 9 booster successfully landing — using only 100% free APIs.

### Features
- XGBoost model trained on Falcon 9 landing attempt
- Real-time marine weather forecast via Open-Meteo 
- Pure FastAPI backend + vanilla HTML/CSS/JS + Plotly frontend (no React, no Streamlit)
- Deployed live 24/7 on free tier (Render/Railway)
- 2010–2020 historical model (IBM dataset)  
- Live next-launch integration (real mission, real countdown, real wind)  
- Designed & built by Azam Effendi



### Tech Stack (all free)
FastAPI • XGBoost • Open-Meteo • Plotly.js • GitHub Pages / Render
